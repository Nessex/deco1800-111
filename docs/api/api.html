/user

/prompts

Get a list of prompts with associated details. Defaults to a selection of prompts from multiple tags.

/prompts get

get

get /prompts

  • Request
  • Response

Query Parameters

  • tags: (string)

    comma separated list of tags to search on

  • reactions: (string)

    comma separated list of reactions to search on

HTTP status code 200

Body

Type: application/json

Example:

{
    "success": true,
    "data": {
        "results": [],
    }
}

/stories

Get a list of stories sorted by newest first. Optionally filter results with query parameters.

/stories get

get

get /stories

  • Request
  • Response

Query Parameters

  • tags: (string)

    comma separated list of tags to filter by

  • reactions: (string)

    comma separated list of reactions to filter by

  • author: (string)

    author username to filter results

HTTP status code 200

Body

Type: application/json

Example:

{
    "success": true,
    "data": {
        "results": [],
    }
}
